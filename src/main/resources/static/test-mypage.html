마이페이지 API 테스트

서버: http://localhost:8080 | 로그인 먼저 해야 인증이 필요한 API가 동작합니다.

0. 소셜 로그인

1. 내 정보 조회 GET /api/v1/users/me

2. 프로필 수정 PATCH /api/v1/users/me/profile

3. 월 예산 설정 PATCH /api/v1/users/me/budget

4. 알림 설정 PATCH /api/v1/users/me/notification-settings

5. 소비 통계 GET /api/v1/users/me/stats

6. 소비 기록 GET /api/v1/users/me/wishes/history

7. 나의 게시글 GET /api/v1/users/me/posts

8. 내가 투표한 게시글 GET /api/v1/users/me/votes

9. 약관 목록 GET /api/v1/terms

10. 약관 단건 GET /api/v1/terms/{type}

11. 계정 탈퇴 DELETE /api/v1/users/me

⚠ 모든 데이터가 삭제됩니다. 신중하게 누르세요.
